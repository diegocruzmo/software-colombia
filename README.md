This is a technical test in Next.js for Software Colombia.

This project uses:
Tanstack Query
Shadcn UI
Tailwind CSS
Axios
TS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
